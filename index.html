About

Yoo's Game: Maze Runner adalah game labirin 2D yang menguji kemampuan navigasi dan strategi pemain. Kumpulkan semua koin dan temukan pintu keluar untuk menyelesaikan setiap level.

3 Level
10×10 Grid
100% Seru

Features

Labirin Grid

Navigasi melalui labirin berbasis grid dengan dinding, jalan, dan rintangan.

Responsif

Main di desktop dengan keyboard atau di mobile dengan kontrol sentuh.

Multi Level

Tiga level berbeda dengan tingkat kesulitan yang meningkat.

Sistem Skor

Kumpulkan koin untuk meningkatkan skor dan buka pintu keluar.
